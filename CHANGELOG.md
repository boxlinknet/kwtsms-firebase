# Changelog

## [0.1.0] - 2026-03-26

### Added
- Initial release
- Firestore queue trigger for SMS sending
- Auth trigger for welcome SMS on new user signup
- HTTPS callable for on-demand SMS
- OTP generation and verification callable
- Scheduled daily sync (balance, sender IDs, coverage)
- Multilingual template system (English, Arabic)
- Firestore logging with Cloud Functions debug logging
- Test mode support (kwtSMS test=1)
